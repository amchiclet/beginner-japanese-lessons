Survival kanji

You do not need to read kanji to travel. You need to recognise a small set of shapes on signs, the way you recognise a logo. This page collects those shapes as they come up in lessons. Recognition only, no writing required.

How to use this. Don't try to memorise readings. Match the shape to the meaning, like an icon. The romaji is only so you can say it if you need to.

The one trick that reads half the station signs

The character 口 literally means "mouth," and by extension an "opening." So an exit is an out-opening and an entrance is an in-opening:

  • 出 (out) + 口 (opening) = 出口 exit
  • 入 (in) + 口 (opening) = 入口 entrance

Big stations name their exits by compass direction + 口. Learn four compass shapes and you can match "North Exit" on a sign to the one Google Maps told you:1

SignReadingMeans
北口kita-guchiNorth Exit
南口minami-guchiSouth Exit
東口higashi-guchiEast Exit
西口nishi-guchiWest Exit
Tourist move. You rarely need to read these cold. Google Maps gives you "Shinjuku Station, South Exit"; you just match 南口 on the sign to the shape. Recognition, not reading.
