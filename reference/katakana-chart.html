Two marks that reuse these shapes

  • The long dash ー holds the previous vowel longer (カー = kā). Covered in Lesson 2.
  • Dakuten ゛ and handakuten ゜ are small marks that voice a consonant: カ ka → ガ ga, ハ ha → バ ba → パ pa. You don't need to memorise these yet — just know that ゲーム (gēmu) is カ-row カ with the ゛ mark. We'll cover them when we hit those rows.
Recognition first, handwriting optional. If you do want to write them, Tofugu's guide has stroke-order practice sheets. Test recognition on Tofugu's Kana Quiz.
